# Папка `server`

Эта папка содержит серверную часть проекта. Включает в себя:

- `src/` — исходный код сервера.
  - `config/` — конфигурация базы данных.
  - `database/` — миграции и скрипты для заполнения базы данных.
  - `middleware/` — промежуточное ПО (middleware), включая обработку ошибок и авторизацию.
  - `routes/` — маршруты API.
- `package.json` — зависимости и скрипты для запуска сервера.

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2.ngrok http 3000
3. cd C:\Users\nervblyati\Desktop\diplom\server
   npm run db:migrate
4. Настройте переменные окружения в файле `.env`.

5. Запустите сервер:
   ```bash
   npm start
   ```
